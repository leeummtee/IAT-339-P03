Style Guide

INTERACTIVE ELEMENTS

IN-TEXT LINK

CSS selector:

.link

Sample code:

<p> Click <a class="link" href="https://www.sfu.ca/"> here </a> to find out more!/ <p>

Rendered element:

Click here to find out more!

NAVIGATION LINK

CSS selector:

.button-nav

Sample code:

<a class="button-nav" href="#home"> Home</a>

Rendered element:

home

SUBMIT & RESET BUTTONS

CSS selector:

.button-form

Sample code:

<input class="button-form" type="submit" value="Submit">
<input class="button-form" type="reset" value="Reset">

Rendered element

TEXT INPUT FIELD

CSS selector:

.input-text

.input-textlabel

Sample code:

<label class="input-textlabel" for="name"> Name: </label>
<input class="input-text" type="text" id = "name" name="name">

Rendered element:

RADIO BUTTON INPUT FIELD & LABEL

CSS selector:

input

label

Sample code:

<input type="radio" id="elem1" name="type" value="elem1">
<label for="car">element 1</label>

Rendered element:




IMAGE AS A LINK

CSS selector:

.block-projects

.image-link

Sample code:

<div class = "block-projects">
<a href = "https://www.sfu.ca/">
<img class="image-link" src="images/placeholder.jpg" alt="trim cleanse">
</a>
</div>

Rendered element:
